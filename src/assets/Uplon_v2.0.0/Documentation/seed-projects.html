Seed Projects

We have built out the seed project for most of common languages and frameworks to make your job easy when you would like to use the Minton with any of these. We've covered Angular, React, Vue, Laravel, Django, Ruby on Rails and Node so far. As part of each of this seed, we have included structure, layouts, styles and inbuilt authentication, account related pages including registration, password, etc.

In order to get started with any of this seed, please refer to corresponding instructions

Angular Full Version

  1. Go to command line and switch to your seed project directory
  2. Install the dependencies using command npm install. This will install all required dependencies.
  3. Run ng serve for a dev server. Navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files.
  4. Please refer to ReadMe file for more detailed instructions

React Seed

  1. Go to command line and switch to your seed project directory
  2. Install the dependencies using command yarn install or npm install. This will install all required dependencies.
  3. Run yarn start or npm start for a dev server. Navigate to http://localhost:3000/. The app will automatically reload if you change any of the source files.
  4. Please refer to ReadMe file for more detailed instructions

Vue Seed

  1. Go to command line and switch to your seed project directory
  2. Install the dependencies using command yarn install or npm install. This will install all required dependencies.
  3. Run yarn dev or npm dev for a dev server. Navigate to http://localhost:8000/. The app will automatically reload if you change any of the source files.
  4. Please refer to ReadMe file for more detailed instructions

Django Seed

  1. Make sure to have python3, pip3 and virtual environment installed on your system
  2. Install all dependencies using command pip install -r requirements/local.txt
  3. Create database and run migrations using command python manage.py migrate
  4. In order to setup frontend, install the dependencies using command yarn install or npm install
  5. Make sure to have gulp installed and running on your system
  6. Run gulp for starting a dev server. Navigate to http://localhost:8000/. The app will automatically reload if you change any of the source files.
  7. Please refer to ReadMe file for more detailed instructions

Laravel Seed

  1. Make sure to have php (>v7.1), composer and node installed and running on your system
  2. Open up terminal and go to your project directory
  3. Install backend dependencies using command composor install
  4. Install frontend dependencies using command npm install
  5. Migrate database using command php artisan migrate
  6. Compiled all the assets using command npm run dev
  7. Run php artisan serve for starting a dev server. Navigate to http://localhost:8000/.
  8. Please refer to ReadMe file for more detailed instructions

Node Seed

  1. Make sure to have php (>v7.1), mongo installed and running on your system
  2. Open up terminal and go to your project directory
  3. Install all dependencies using command npm install
  4. Run node app.js for starting a dev server. Navigate to http://localhost:8000/.
  5. Please refer to ReadMe file for more detailed instructions

Ruby on Rails Seed

  1. Make sure to have Ruby (> v2.5) and Yarn installed and running on your system
  2. Open up terminal and go to your project directory
  3. Install backend dependencies using command bundle
  4. Install frontend dependencies using command yarn install
  5. Migrate database using command rails db:migrate
  6. Run node app.js for starting a dev server.
  7. Please refer to ReadMe file for more detailed instructions
